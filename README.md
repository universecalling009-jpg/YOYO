# 📚 Amit Books - Android App

A beautiful Android app with Book 1 and Book 2.

---

## 🚀 How to Build APK for FREE using GitHub (No Android Studio needed!)

Follow these steps carefully — it only takes 5 minutes!

---

### STEP 1 — Create a GitHub Account
- Go to 👉 https://github.com
- Click **Sign up** and create a free account

---

### STEP 2 — Create a New Repository
- After logging in, click the **+** button (top right) → **New repository**
- Name it: `AmitBooks`
- Set it to **Public**
- Click **Create repository**

---

### STEP 3 — Upload the Project Files
- On your new repo page, click **uploading an existing file**
- Extract the `AmitBooks.zip` file on your computer
- Drag and drop ALL the extracted files into GitHub
- Click **Commit changes**

---

### STEP 4 — Watch the APK Build Automatically! 🎉
- Click the **Actions** tab at the top of your repo
- You'll see **"Build Amit Books APK"** running automatically
- Wait ~3-5 minutes for it to finish (green ✅)

---

### STEP 5 — Download Your APK
- Click on the completed workflow run
- Scroll down to **Artifacts**
- Click **AmitBooks-APK** to download your APK!

---

### STEP 6 — Install on Your Android Phone
- Transfer the `.apk` file to your Android phone
- On your phone go to: **Settings → Security → Unknown Sources → Enable**
- Tap the APK file to install
- Open **Amit Books** and enjoy! 📱

---

## App Features
- 📚 Home screen with Amit Books branding
- 📖 Book 1 button (opens Book 1 content)
- 📗 Book 2 button (opens Book 2 content)
- ← Back button to return home
- Warm brown/beige book theme
